# Responsive Minimalist Footer - CodingLab

Um rodapé de site (Footer) moderno, minimalista e totalmente responsivo desenvolvido com HTML5 e CSS3. O projeto apresenta uma estrutura clássica em três colunas principais sobre um fundo escuro (*Dark Mode*), integrando textos institucionais, links de serviços, formulário de newsletter e ícones de redes sociais.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica das seções de rodapé, listas de links e formulário de captura.
* CSS3: Estilização com esquema de cores escuras, tipografia sem serifa e customização de botões.
* Flexbox / CSS Grid: Distribuição e alinhamento responsivo das três colunas principais na tela.

## Funcionalidades e Técnicas Aplicadas

* Estrutura Organizada em Três Colunas:
  * **About us & Contact us:** Seção dedicada a uma breve descrição do canal/empresa e exibição de dados de contato (telefone e e-mail).
  * **Our Services:** Lista de links limpa e vertical contendo serviços como Mobile Application Design, Wireframing & Prototyping, Web Design e UX/UI Design.
  * **Subscribe us:** Formulário focado em conversão com campo de entrada de e-mail e um botão de ação "Send" em vermelho vibrante de alto contraste.
* Ícones de Redes Sociais: Bloco inferior com botões minimalistas quadrados para direcionamento (Facebook, Twitter, Instagram, YouTube e LinkedIn).
* Barra de Direitos Autorais: Linha de base discreta com a menção de Copyright integrada à identidade visual da "CodingLab".
